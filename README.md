# C-EXEC

Some helper functions for launching programs in Unix machines.

+ *sys/pathsearc.h* : Search files in a path.           [man](doc/pathsearch.3.md)
+ *sys/forkexec.h*  : Wrapper around *fork()* *exec()*.
